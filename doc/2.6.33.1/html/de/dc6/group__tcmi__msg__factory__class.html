Detailed Description

This <<singleton>> class is responsible for message materialization based on message ID.

In addition it provides an optional verification whether the specified message ID belongs to the requested message group.

Function Documentation

<<public>> Materializes the message.

Initially, a verification is performed whether the message ID belongs to the requested message group. This way, unexpected messages from a wrong message group won't get materialized.

Once a proper message descriptor is retrieved, the message creation is delegated to the protocol message super class

Parameters
msg_id- Full identifier of the message including flags.
msg_group- this is for verification, when the factory user wants to make sure the message comes from a particular message group
Returns
message instance or NULL
