static int tcmi_task_copy_strings ( char ***  dst,
char **  src 
)
inlinestatic

<<private>> Class method - copies an array of strings terminated by NULL string (standard argv/envp format) into the destination array.

Destination array are released and new memory for the strings is allocated. The idea how the array in the task will be organized is sketched below.

Parameters
***dst- pointer(*) to the destination array(**) (should be &self->argv or &self->envp
**src- source array of strings
Returns
0 upon success
