35 /** @defgroup tcmi_rpc_class tcmi_rpc class
36  *
37  * @ingroup tcmi_syscall_group
38  *
39  * A <<virtual>> RPC class, used as parent for both guest and shadow side.
40  *
41  * This class is used for calling methods on shadow on guest request. If guest task calls RPC,
42  * the remote side is executed by coresponding shadow task. Communication is done using
43  * \link tcmi_rpc_procmsg tcmi_rpc_procmsg class \endlink for RPC request and
44  * \link tcmi_rpcresp_procmsg tcmi_rpcresp_procmsg class \endlink for response.
45  *
46  * @{
47  */
