Detailed Description

Classes in this group are used for system calls modification and forwarding to provide migration transparency.

Each class is divided into parts with related system calls:

  • PID & GID manipulation - These methods are used for PID and GID manipulation or works with sessions
  • User identification - Methods from this part works with user identification of current process
  • Group identification - Methods from this part works with group identification of current process
  • Signal sending - These methods are used for sending signals
  • Generic - Methods from this part are able to handle more calls from diferent groups. Only tcmi_guest_rpc has this part.
