Detailed Description

This message is send by PEN or CCN when it is going to disconnect from its peer.

It serves as a notification, to tell the peer it shall as well terminate the connection. In addition, if send from CCN to PEN, the PEN should as a resonpse first try to migrate back all tasks from a peer core node.
