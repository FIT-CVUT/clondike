39 /** @defgroup tcmi_ckptcom_class checkpointing component
40  *
41  * @ingroup tcmi_module_class
42  *
43  * <<Singleton>> class that creates a checkpoint of a process.
44  * It uses the checkpoint class as a temporary object to store
45  * the state of the process.
46  *
47  * It provides a simple interface that that allows:
48  * - creating a checkpoint into a specified file
49  * - restoring the checkpoint via a new bin_fmt handler that it
50  * registers in the kernel.
51  *
52  * While creating a checkpoint there are two types of checkpoints
53  * supported - heavy and light. See tcmi_ckptcom_checkpoint() for
54  * explanation.
55  *
56  * @{
57  */
