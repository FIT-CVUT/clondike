Go to the documentation of this file.
1 1. Install the necessary tools for kernel compilation:
2  1.1 ncurses-dev
3  1.2 gcc
4  1.3 make
5  1.4 initramfs-tools
6  1.5 git
7 
8  It can be done with one command "apt-get install ncurses-dev gcc make initramfs-tools git" as root.
9 
10 2. Download Clondike patch from github.com:
11  https://github.com/MarstCVUT/Clondike.git (manually link Downloads at right).
12 
13 3. Download the appropriate kernel from www.kernel.org/pub/linux/kernel/v2.6/linux-2.6.33.1.tar.gz.
14  You can use commend "wget www.kernel.org/pub/linux/kernel/v2.6/linux-2.6.33.1.tar.gz".
15 
16 4. Unpack downloaded kernel i.e. by command
17  "tar -xvf linux-2.6.33.1.tar.gz"
18 
19 5. Go to the unpacked directory i.e.
20  "cd linux-2.6.33.1".
21 
22 6. Make a new kernel patch by command
23  "patch -p1 < path_to_the_downloaded_patch" (clondike_kernel_2.6.33.1.patch)
24 
25 7. If we've old and functional kernel config file, for example config-2.6.32-5-xen-amd64, we can use it.
26  Rename it to actual config file .config and compile it:
27  "cp config-2.6.32-5-xen-amd64 .config"
28  "make oldconfig"
29 
30 8. Enable in kernel settings several required properties.
31  Use spacebar to change configuration in menu, after all configurations save it to .config
32  by Save an Alternate Configuration File and Exit.
33 
34  "menuconfig"
35  menu: Clondike > Enable TCMI Enable TCMI CCN Enable TCMI PEN
36  menu: File systems/Network File systems/NFS client support (ENABLE *)
37  > Enable ROOT_NFS (jeste doresit zavislosti viz hledani /root_nfs)
38 
39  menu: File systems/Network File Systems > Enable Plan 9 Resource Sharing Support
40  menu: Networking support/Networking options > Disable Appletalk protocol support
41  menu: Networking support > Enable Plan 9 Resource Sharing Support
42 
43 9. Copy recursively a directory Clondike/src to the directory clondike i.e.
44  "cp -R ../Clondike-master/src/ clondike/".
45 
46 10. The kernel compilation (in a directory linux-2.6.33.1)
47  "make clean"
48  "make -j4"
49  "make modules"
50  "make modules_install"
51  "make install"
52 
53 11. Go to directory /boot "cd /boot"
54 
55 12. There create your own ramfs by command mkinitramfs
56  "mkinitramfs -o initrd.img-2.6.33.1-clondike-nfs 2.6.33.1-clondike-nfs"
57 
58 13. GRUB: Add a line proxyfs.server=tcp:0.0.0.0:1112 to a grub config file
59  "echo "proxyfs.server=tcp:0.0.0.0:1112" >> /etc/default/grub"
60  GRUB2: Add GRUB_CMDLINE_LINUX="proxyfs.server=tcp:0.0.0.0:1112"
61 
62 14. Erase files with old indentity in the director
63  "rm * /root/clondike/userspace/simple-ruby-director/conf"
64 
65 15. Reboot system a load right kernel linux-2.6.33.1
66  "reboot"
67 
68 16. Install ruby on the new system
69  "apt-get install ruby"
70 
71 17. Compile npfs server for function 9p sharing filesystem:
72  NPFS is in folder /root/npfs_install
73  17.1. Compile NPFS command "make"
74  17.2. Execute file /root/npfs_install/fs copy to folder /root/npfs
75  17.3. Execute script /root/npfs/npfs-start &
76  17.4. Copy init script for NPFS from /etc/init.d/npfsd to same location in system
77  for clondike /etc/init.d/rc.local to same location in system and overwrite it
78 
79  Single commands:
80  # mkdir /root/npfs
81  # /root/clondike/root/npfs_install/fs/npfs /root/npfs/
82  # /root/clondike/root/npfs/npfs-start /root/npfs/
83  # chmod +x /root/npfs/npfs-start
84  # /root/npfs/npfs-start &
85  # cp /root/clondike/etc/init.d/npfsd /etc/init.d/
86  # cp /root/clondike/etc/init.d/rc.local /etc/init.d/
87 
88 18. Registered script for npfsd into INIT process by command:
89  # update-rc.d npfsd defaults
90 
91 19. Execute script
92  "/root/clondike/scripts/install.sh"
93 
94 20. Restart director to generate new unique certificate
95  "/root/clondike/scripts/restart-director.sh"
