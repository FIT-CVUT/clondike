int msg_transaction_do ( int  msg_code,
struct msg_transaction_ops *  ops,
void *  params,
int  interruptible 
)

Performs a netlink transaction.

1) Creates a message (with specified code) 2) Uses "create_resquest" method to fill params 3) Sends the request 4) Reads response 5) Uses "read_response" to parse output data

Parameters
msg_codeCode of the message to be created
opsTransaction specific operations to fill params and read response
paramsStructure that will contain input params and hold output values
interruptibleif !=0, the waiting for response can be interrupted by a signal
Returns
0 on success, error code otherwise
