Called, when a detached node connects to the core node.

Parameters
addressProtocol specific address string of a detached node
slot_indexIndex in manager that was assigned to this connection
auth_data_sizeSize of authentication data, if any
auth_dataAuthentication data, or NULL
acceptOutput parameter. 0 if remote node connection should be rejected, otherwise it is accepted
Returns
0 on success, error code otherwise. In case of error, output params are not valid!
