35 /** @defgroup tcmi_shadowtask_class tcmi_shadowtask class
36  *
37  * @ingroup tcmi_task_class
38  *
39  * This class is an abstraction for a migrated process on CCN. It
40  * represents the residual dependency that the migrated process has
41  * towards its originating node(CCN). This class also does all the
42  * migration work including migration back to CCN (merging the shadow
43  * with the process migrating home). For this purpose, there are
44  * various methods that implement various types of migration:
45  * - preemptive migration using physical checkpoint image
46  * (tcmi_shadowtask_*_ppm_p())
47  * - preemptive migration using virtual checkpoint image
48  * (tcmi_shadowtask_*_ppm_v())
49  * - non-preemptive migration via execve hook
50  * (tcmi_shadowtask_*_npm())
51  *
52  * @{
53  */
