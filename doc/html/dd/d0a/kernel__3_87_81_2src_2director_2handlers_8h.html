Handler of user message transmission requests.

Parameters
is_core_node1 if we shall send to core node manager, 0 if we shall send to detached node manager
target_slot_indexSlot of a manager to sent message to
Returns
Can return (error return codes go back to user space in error field of ack):

ENOENT - Invalid slot_index -> No such a peer ENOMEM - Run out of memory EFAULT - Failed to send message for whatever reason
