Request to immigrate process from core node specified by slot_index to this node.

Parameters
slot_indexIndex of the remote core node manager that requests immigration to this node
uidUid of user on the remote node that is requesting the migration
nameName of the binary the process (executable) that should be immigrated
acceptOutput param. 0 if immigration is rejected, everything else if it is accepted
Returns
0 on success, error code otherwise. In case of error, output params are not valid!
