Data Structure Documentation

struct tcmi_npm_params

Structure containing data required for performing of non-preemtive restart.

The structure contains all required data in itself, the pointers all point to a "data" field. The organization in data field is as follows: args array (pointer to strings) + null termination args[0] first arg + null termination ..... envp array (pointer to envp strings) + null termination ...
