35 /** @defgroup tcmi_guesttask_class tcmi_guesttask class
36  *
37  * @ingroup tcmi_task_class
38  *
39  * This class is an abstraction for an 'immigrated' process on PEN. It
40  * represents the migrated process, so that the TCMI framework is able
41  * to control it.
42  *
43  * This class also does all the migration work including migration back
44  * to CCN. For this purpose, there are various methods that implement
45  * various types of migration:
46  * - preemptive migration using physical checkpoint image
47  * (tcmi_guesttask_*_ppm_p())
48  * - preemptive migration using virtual checkpoint image
49  * (tcmi_guesttask_*_ppm_v())
50  * - non-preemptive migration via execve hook
51  * (tcmi_guesttask_*_npm())
52  *
53  * @{
54  */
