static kkc_arch_obj_t kkc_sock_tcp_new ( void  )
static

<<private>> Creates an empty TCP socket.

The BSD socket is created for a stream connection using TCP protocol. In addition, socket reuse option is set. Eventhough a regular TCP socket instance is created it is casted to kkc_arch_obj_t, which is a generic object supported by kkc_arch. The user of the architecture class instance always knows, what instance to expect, so he/she can cast it to kkc_sock without problems. The reason, why we cannot return directly kkc_sock, is that it would create a circular dependency between kkc_arch and kkc_sock .

Returns
architecture object instance or NULL
