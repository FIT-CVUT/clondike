Detailed Description

  • main TCMI module 
                 - on module init:
                    initializes the TCMI CCN manager component
                    initializes the TCMI PEN manager component
                 - on module unload: 
                    shutsdown the TCMI CCN manager component
                    shutsdown the TCMI PEN manager component
