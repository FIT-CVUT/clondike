This is the main entry point that can be used by TCMI when it needs to consult user space director.

Checks, whether the process should be non-preemptively migrated to some other node.

Parameters
pidPid of the process
uidEffective user ID of the process being executed
is_guest1 if the process is guest
nameFile name, that is being executed
argvArgs of execve
envpEnvp of execve
migman_to_useOutput parameter.. slot index of migration manager to be used
migrate_homeOutput param .. should we migrate home?
rusageResource usage structure of process
Returns
0 on success, if no migration should be performed 1 on success, when a migration should be performed error code otherwise. In case of error, output params are not valid!
