Detailed Description

This is a helper class used by the checkpoint to keep track of already open file descriptors.

It allows easy lookup of a duplicate file descriptor. This is needed when checkpointing open files of a process.

There is no need for locking nor reference counting as fdcache is a temporary object (as well as a checkpoint object ) accessed from a single thread of execution. The reference counter is just to keep it consistent with current software design.
