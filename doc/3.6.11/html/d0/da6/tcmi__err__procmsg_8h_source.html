36 /** @defgroup tcmi_err_procmsg_class tcmi_err_procmsg class
37  *
38  * @ingroup tcmi_procmsg_class
39  *
40  * This class represents a generic error process message. It carries an error
41  * code and can be used as response to any regular process message that has
42  * registered its rx constructor as the error rx constructor with the
43  * message factory. Unlike other messages, it doesn't define a fixed
44  * message ID. Instead the transfer message constructor
45  * (tcmi_err_procmsg_new_tx()) allows specifying a message ID that would
46  * carry the regular response message.
47  *
48  * The constructor only sets error flags in the message ID. The
49  * receiving party identifies the message based on the regular
50  * response ID. The message flags indicating an error make \link
51  * tcmi_msg_factory_class the message factory \endlink use the error rx
52  * constructor. After delivery, the error message can be processed
53  * and the error code extracted.
54  *
55  * @{
56  */
57
