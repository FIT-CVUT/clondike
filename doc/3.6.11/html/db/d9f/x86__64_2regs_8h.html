Data Structure Documentation

struct tcmi_x86_64_regs

Copy of corresponding pt_regs struct in ptrace.h.

Must be kept in sync with kernel pt_regs!

The reason why we make a copy here is, that we want to use multiple pt_regs concurrently (of multiple platforms), but this is not possible with standard pt_regs, so we create our name-distinguished copy. We also have to modify the types of the registers, to use platform neutral types.
