# Coding convention #

## Naming ##

### Class Names ###

    UpperCamelCase

### Methods ###

    lowerCamelCase

### Class instance variables ###

    @lowerCamelCase

### Constants ###

    CAPITALIZED_WITH_UNDERSCORES

## Indentation ##

We use tabulators!

## Other ##

For other convetions see the standard Ruby convetion defined in [The Ruby Style Guide](https://github.com/bbatsov/ruby-style-guide)
